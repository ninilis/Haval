# Тестовое задание HTML-верстальщик
Лендинг рекламной кампании автомобилей HAVAL. Выполнен по макету из Figma.  
Адаптирован под мобильные устройства (планшеты, смартфоны).

## Демо-версия
[GitHub Pages](https://ваш_логин.github.io/haval-test/).

## Стек
- **HTML5** (семантика, SVG-графика)
- **CSS3** (Flexbox, Grid, CSS-переменные, анимации, media queries)
- **JavaScript (Vanilla)** – переключение цветов авто, таймер, модальное окно
- **Sinoby Timer** – встраиваемый обратный отсчёт (пресет rings-soft)
- **Шрифты** – Haval Regular / Medium

## Что внутри
- Две карточки автомобилей (Haval M6, Haval Dargo X)
- Смена цвета машины по клику на кружки
- Плашка «В наличии» с наложением на карточку
- Таймер обратного отсчёта с настройками
- Всплывающее окно «Оставить заявку» (поле телефона + отправка)
- Адаптивная вёрстка (планшеты, телефоны)

## Затраченное время
На выполнение ушло около **17 часов**.  
Основные трудности: точное позиционирование маски и плашки, адаптация таймера.
